# Contemporary India - Education Studies Documentation

This repository contains comprehensive study material on Contemporary India and its Educational Systems, organized as a MkDocs-powered documentation website.

### Prerequisites
- Python 3.7 or higher
- pip (Python package installer)

### Installation

1. Install the required packages:
```bash
pip install -r requirements.txt
```

### Running the Documentation Locally

To view the documentation in your browser with live reload:

```bash
mkdocs serve
```

Then open your browser and navigate to `http://127.0.0.1:8000`

### Building the Documentation

To build the static site:

```bash
mkdocs build
```

The built site will be in the `site/` directory.

### Deploying to GitHub Pages

To deploy the documentation to GitHub Pages:

```bash
mkdocs gh-deploy
```
